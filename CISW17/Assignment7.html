Lesson 16: Understanding the Importance of Responsive Web Design

Responsive web design (RWD) is an approach to web design that considers the customers device and adapts the design for optimal viewing. RWD helps web designers and site owners by creating an “edit once, display anywhere” website


Responsive web design is a combination of many design techniques, including the following:

  • CSS—especially media queries
  • Cleans
  • valid HTML code
  • Progressive enhancement
  • Feature device
  • browser detection using scripts
  • Server-side components to produce faster sites

    Lesson 17: Designing for Mobile Devices

    The main reason mobile design is important is because more and more people are using mobile devices to access the web.Mobile First is primarily an implementation of progressive enhancement. You focus on getting the required content and functionality to as many customers as possible and then enhance the site for more diverse devices.

    Traditionally, a web designer first built a website for desktop users and then modified that site for mobile users. Mobile First design turns that formula on its head: It says to build a site for mobile users first and then adjust it to work for desktop users.Two aspects of websites that can be difficult to make responsive are tables and images. Data tables are often very big and can thus be very difficult to handle in a responsive design, especially for small devices.

    Lesson 18: Using Media Queries and Breakpoints

    According to the W3C, a media query is a “logical expression that is either true or false. A media query is true if the media type of the media query matches the media type of the device.
    There are 10 media types you can test for with CSS media queries:

  • all—All media
  • braille—Braille and tactile feedback devices
  • embossed—Paged braille printers
  • handheld—Small-screen low-bandwidth handheld devices
  • print—Paged media and documents in print preview mode
  • projection—Projected presentations
  • screen—Color computer screens
  • speech—Speech synthesizers
  • tty—Teletypes and media with a fixed-pitch character grid
  • tv—Television

    • A CSS breakpoint is a device feature with a media query declaration assigned to it. They are most commonly based on browser widths, so most designers think of breakpoints as places where the width of the design changes to accommodate different devices.Defining breakpoints is done with CSS media queries. The most common type of breakpoint is based on the width of the device.Media query expressions are the basis of responsive web design.
